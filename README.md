# nvim
My nvim config


## Setup

1. Clone the repository
```
git clone git clone https://github.com/gavinaboulhosn/nvim.git
```

2. Install Packer

See [QuickStart](https://github.com/wbthomason/packer.nvim#quickstart)

3. Sync dependencies with :PackerSync
```
cd ~/.config/nvim
nvim .
:PackerSync
```

4. Profit
